About

Prior to becoming a full stack web developer, I was studying Marketing at CUNY Baruch. After working at several different companies, I realized that I was unsatisfied with the work I was producing and wanted something more meaningful and impactful.

In 2022, I decided to switch careers and pursue web development. My favorite part of web development is building practical solutions to real-life problems and understanding that every line of code plays a valuable role in creating the finished product.

When I'm not coding, you can most likely find me at the gym, playing basketball, or exploring new places!

Projects

Discovr
Live Link GitHub

Discovr is a clone of Flickr built using React/Redux for the frontend, Flask for the backend API.

Discovr is a full-stack clone of Flickr. Users can log in/sign up to explore photos taken by users from all around the world! Logged-in users can add new photos to the explore page, like, and leave comments on any photos! Find inspiration for your next travel destination and Discovr the world!

Airdnd
Live Link GitHub

Airdnd is a clone of Airbnb built using React/Redux for the frontend, Sequelize for the backend API.

Airdnd is a full-stack clone of Airbnb. Users can log in/sign up to find their next destination! Logged-in users can create a new spot, add reviews, and create a new booking for an existing spot. Owners of the spot can also add additional images to show off their homes! Manage your bookings and check for current bookings on the details page to plan your next trip!

Gulped
Live Link GitHub

Gulped is a clone of Yelp built using React/Redux for the frontend, Sequelize for the backend API.

Gulped is a full-stack clone of Yelp, where users can log in/sign up to become local food critics! Business owners can add their business, add menu items, and leave reviews on other businesses' pages. In the mood for something specific? Tell the search bar at the top what you have in mind and we'll show the best spots around town for it!
